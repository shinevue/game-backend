## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add rate_limit
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"rate_limit": {}
}

```
</Tab>

</Tabs>
