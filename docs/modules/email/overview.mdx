## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add email
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"email": {
		"config": {
			// Your config here. See below for more details.
		}
	}
}

```
</Tab>

</Tabs>

## Config

### Schema
		
```typescript
export type Provider =
  | {
      test: ProviderTest;
    }
  | {
      sendGrid: ProviderSendGrid;
    };

export interface Config {
  provider: Provider;
}
export interface ProviderTest {}
export interface ProviderSendGrid {
  apiKeyVariable?: string;
}
```
